## Clicks Analysis

### Pre-requistes

1) Install nvm

	```
	curl https://raw.githubusercontent.com/creationix/nvm/v0.25.0/install.sh | bash
	```

2) Install node version

	```
	nvm install 8.94
	```

3) Use node version
	
	```
	nvm use 8.9.4
	```

### Run the Application
	
	npm install
	npm run solution	

### Run test cases
	
	npm run test

### Library used

1) momentjs: To perform Date operations
2) mocha: To perform unit test cases
